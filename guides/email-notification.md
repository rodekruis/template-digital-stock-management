## Optional feature

Prerequisites:

- Published dashboard on a workspace (including sematic model)
- Install [Power BI Report Builder](https://learn.microsoft.com/en-us/power-bi/paginated-reports/report-builder-power-bi)
- Access to [Power Automate environment](https://make.powerautomate.com/), preferrably with the same subscription of Power BI
- Install [Power Automate](https://learn.microsoft.com/en-us/power-automate/desktop-flows/install)
